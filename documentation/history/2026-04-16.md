# 2026-04-16

## Accomplished

- Added dedicated `skipUnchangedImages` setting (default: true) to enable/disable unchanged image skip optimization independently of `overwriteExisting` setting.

## Key Decisions

- `skipUnchangedImages` check takes precedence in `transcribeFile()`; if enabled and image is unchanged, it skips regardless of `overwriteExisting`.
- Default value is `true` to maintain existing optimization behavior for users upgrading.
- Uses existing transcription cache mechanism (file fingerprint + config signature).

- `TranscriptionService.transcribeFile()`: refactored skip logic to check `skipUnchangedImages` first, then `overwriteExisting`
